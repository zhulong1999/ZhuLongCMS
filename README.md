## 安装系统：

1. 访问 当前URL/install
2. 阅读许可协议并同意
3. 开启相关PHP扩展或目录权限
4. 填写数据库相关信息并且设置管理员初始密码
5. 最后点击安装

## 进入系统

- 访问当前URL/admin
- 输入管理员账号 默（ 账号：admin 密码：admin123}
- 点击登录
